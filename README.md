## Controls
Button | Action
-------|--------
Mouse move | Look around
Left mouse button | Break block
Right mouse button | Place block
W, A, S, D | Move around
Space/Double space | Jump/Fly up
Left Control | Sprint
Left shift | Sneak (coming soon)/Fly down
Escape | Lose window focus
F2 | Take screenshot
F3 | Show/Hide game infos
